# Saturno do Amor

**Um universo de declarações.** Este projeto cria uma cena 3D interativa com um planeta central rodeado por anéis de mensagens românticas flutuantes.

## Como Usar

1. Abra o arquivo `index.html` em qualquer navegador
2. **Interaja**: Clique e arraste (ou toque e arraste no celular) para rotacionar a câmera ao redor do planeta
3. **Observe**: As mensagens nos anéis giram em velocidades diferentes, com opacidade variando conforme a posição

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Versão | Finalidade |
|------------|--------|-------------|
| **Three.js** | r128 | Renderização 3D WebGL |
| **HTML5 Canvas** | - | Geração dinâmica de texturas e sprites |
| **JavaScript (ES6)** | - | Animações, controles de câmera e sistema de partículas |
| **WebGL** | - | Aceleração gráfica por hardware |
